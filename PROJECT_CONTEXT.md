# 项目上下文摘要

## 产品概述
- **项目名称**: AI 学习规划师
- **产品定位**: 基于大模型和多智能体系统的个性化学习规划平台
- **目标用户**: 个人学习者 + 学生群体
- **核心价值**: 通过 AI 对话理解需求，生成可执行的学习计划，配合游戏化激励

## 技术栈
- **后端**: Python 3.11 + FastAPI
- **前端**: UniApp + Vue3
- **数据库**: PostgreSQL + Redis
- **AI**: 阿里云 DashScope (通义千问)
- **云服务**: 阿里云

## 关键约束
- 后端必须 Python
- 偏好阿里云服务
- 初期种子用户优先

## 会员模式
- 月度订阅: 100,000 Token
- 年度订阅: 1,000,000 Token

## 等级体系
- 入门: 0-499 积分
- 进阶: 500-1999 积分
- 大师: 2000-4999 积分
- 宗师: 5000+ 积分

## 核心 Agent
1. Router Agent - 意图识别
2. Goal Agent - 学习目标理解
3. Plan Agent - 计划生成
4. Feedback Agent - 评估反馈
